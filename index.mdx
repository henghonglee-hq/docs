GnosisRamp synchronises capability routing, compliance orchestration, and money movement into a single API. The platform handles multi-provider coverage, automatically selecting the best provider for each transaction.

  <Card title="Deterministic routing" icon="route">

## Platform capabilities

| Layer | What it does |
| --- | --- |
| Capability routing | Evaluates source and destination accounts and selects the optimal provider for the transaction. |
| Compliance management | Orchestrates compliance requirements, manages onboarding workflows, and tracks requirement statuses. |
| Money movement | Creates quotes, executes transfers, and provides deposit instructions for customers. |
| Events | Sends webhook events for intent updates, compliance changes, and transaction status with HMAC-SHA256 signatures. |

The platform provides consistent routing, secure credential management, and structured event tracking across all providers and payment rails.
